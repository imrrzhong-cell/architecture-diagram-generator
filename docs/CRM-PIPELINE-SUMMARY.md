## 🚀 如何使用

### **立即查看架构图**
```bash
# 在浏览器中打开
open /Users/rustyryan/crm-architecture-renderer.html
```

### **导出为 PDF**
1. 打开 HTML 文件
2. 点击右上角"导出为 PDF"按钮
3. 在打印对话框中选择"另存为 PDF"
4. 建议设置：
   - 纸张大小：A3
   - 方向：横向
   - 启用"背景图形"

### **导出为 PNG**
1. 打开 HTML 文件
2. 使用浏览器开发者工具（F12）
3. 右键点击架构图 → "捕获节点截图"
